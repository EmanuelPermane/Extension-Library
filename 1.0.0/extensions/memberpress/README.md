# MemberPress

Version: 1.0.0

This extension integrates Manny REST Route Manager with MemberPress.
Integrates MemberPress memberships and rules into Manny entitlements.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- memberpress/memberpress.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
