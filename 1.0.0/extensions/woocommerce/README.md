# WooCommerce Integration

Version: 1.0.0

This extension integrates Manny REST Route Manager with WooCommerce Integration.
Order and subscription entitlements backed by WooCommerce.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- woocommerce/woocommerce.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
