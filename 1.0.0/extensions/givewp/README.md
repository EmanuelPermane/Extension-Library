# GiveWP

Version: 1.0.0

This extension integrates Manny REST Route Manager with GiveWP.
Integrates GiveWP donation history into Manny entitlements.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- givewp/givewp.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
