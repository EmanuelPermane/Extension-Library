# AffiliateWP Integration

Version: 1.0.0

This extension integrates Manny REST Route Manager with AffiliateWP Integration.
Affiliate accounts, referrals, and entitlements powered by AffiliateWP.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- affiliate-wp/affiliate-wp.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
