# Restrict Content Pro Integration

Version: 1.0.0

This extension integrates Manny REST Route Manager with Restrict Content Pro Integration.
Membership plans and access flags via Restrict Content Pro.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- restrict-content-pro/restrict-content-pro.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
