# Easy Digital Downloads

Version: 1.0.0

This extension integrates Manny REST Route Manager with Easy Digital Downloads.
Integrates Easy Digital Downloads purchases and licenses into Manny entitlements.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- easy-digital-downloads/easy-digital-downloads.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
