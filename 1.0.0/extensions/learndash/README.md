# LearnDash

Version: 1.0.0

This extension integrates Manny REST Route Manager with LearnDash.
Integrates LearnDash courses and progress into Manny entitlements.

Requirements:
- Manny REST Route Manager 1.0.0 or newer.
- sfwd-lms/sfwd_lms.php active on this site.\n\nSee the Manny documentation for installation and configuration details.
